O enredo do shell é um programa escrito na linguagem de comandos do shell. Um script da cápsula é um arquivo que consiste em uma seqüência de comandos e é de fato uma sofisticada linguagem interpretativa de programação, com sua própria estrutura de variáveis, de expressões, de seqüência, de decisão e de repetição.

2.2.5. Arquivos e Diretórios no UNIX

Um arquivo UNIX é uma seqüência de 0 ou mais bytes, contendo informações arbitrárias. Nenhuma distinção é feita entre arquivos ASCII, arquivos binários, ou qualquer outro tipo de arquivo.
Por convenção, muitos programas esperam receber nomes de arquivos compostos por um nome-base seguido de uma extensão, separados por um ponto. Esta regra de construção de nomes não é imposta pelo sistema operacional.
Os arquivos podem ser agrupados em diretórios, por conveniência de seus usuários. Os diretórios são armazenados como se fossem arquivos, e, de uma forma geral, podem ser tratados como arquivos. Os diretórios podem conter subdiretórios, levando o sistema de arquivos a exibir uma estrutura hierárquica.

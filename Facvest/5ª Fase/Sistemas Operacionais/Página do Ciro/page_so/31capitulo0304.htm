2.0. ARQUIVOS

Nesta seção vamos descobrir como os arquivos são usados e quais as suas propriedades.

2.1. Identificação de Arquivos

Arquivos são um mecanismo de abstração que fornecem uma forma de armazenar e recuperar informações em disco. Isto deve ser feito de uma tal forma que mantenha o usuário isolado dos detalhes a respeito de como as informações são armazenadas, e de como os discos efetivamente trabalham.
Quando um processo cria um arquivo, é preciso que tal arquivo receba um nome, normalmente dado pelo processo. Quando tal processo termina sua execução, o arquivo continua a existir, podendo ser acessado por outros processos, usando para tanto o nome atribuído ao arquivo.
As regras para identificação de arquivos de sistema, mas todos os sistemas operacionais aceitam como nomes de arquivos válidos conjuntos de um a oito caracteres. Então andréa, bruce e cathy são nomes possíveis para se identificar arquivos. Freqüentemente os dígitos e os caracteres especiais são admitidos, de forma que nomes como 2, urgent! e fig. 2.14 muitas vezes também são válidos.
Alguns sistemas de arquivos fazem distinção entre letras maiúsculas e minúsculas, Por exemplo, o UNIX. Então, no UNIX os arquivos a seguir identificados são todos distintos: bárbara, Bárbara, BARBARA, BÁRBARA e BarRaRa. No MS-DOS todos estes nomes identificam o mesmo arquivo, da mesma forma que o impresso mostrado abaixo, emitido por computador.
A maioria dos sistemas operacionais suporta nomes de arquivos compostos de duas partes separadas por um ponto, como prog.c.

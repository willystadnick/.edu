2.3. CONCEITOS FUNDAMENTAIS DO UNIX

Vamos ver alguns conceitos básicos suportados pelo UNIX, especialmente processos, memória, o sistema de arquivos, e entrada/saída. Estas noções são muito importantes, pois as chamadas de sistema – a verdadeira interface do sistema operacional – fazem uso freqüente destes conceitos.

2.3.1. Processos do UNIX

Os processos são as únicas entidades ativas do sistema UNIX, ele tem apenas um contador de programa, que cuida de guardar o endereço da próxima instrução a ser executada.
O UNIX é um sistema que suporta multiprogramação (paralelismo aparente). Nos sistemas, até quando o usuário está ausente, existem diversos processos rodando em background.
Um destes processos é o chamado cron. Ele acorda uma vez por minuto para verificar se existe alguma tarefa para ser realizada por ele. Se houver, ele faz o trabalho e vai dormir até chegar o momento da próxima verificação.
Este processo é necessário, pois é possível no UNIX agendar atividades a serem realizadas nos próximos minutos, horas, dias, ou mesmo meses.

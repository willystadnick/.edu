As maquinas 386,486 e Pentium não podem usar a memória estendida, ela é então usada para preencher buracos nas áreas de memória superior.Exata união hoje é conhecida como LIMEMS (Lótus Intel Microsoft Expanted Memory System ).
Uma das tarefas da memória expandida e tentar resolver e assentar programas e estruturas de dados grandes em um espaço virtual de 640K.
A memória expandida é totalmente visível ao programador, ou no mínimo para o compilador.Em resumo o modelo de memória do MS DOS e extremamente complicado, com 6 técnicas diferentes q são utilizados para romper a barreira dos 640K e fazer o uso de memória estendidas. São elas:
HMA: move o MS DOS para fora dos 640K
Overlay: Permite que grandes programas rodem dividindo-os em pedaços
Área de memória superior: Recebe o MS DOS , drives de dispositivos e programas TSR
Memória Expandida: Permite que grandes conjuntos de dados possam ser usados, mapeaando-os para dentro e fora da área de memória estendida.
Ram do disco: Utiliza a memória estendida como se fosse um disco de memória.
Cachê:Utiliza a memória estendida colocando um buffer cachê lá.

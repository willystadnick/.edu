CONCLUSÃO

Com o término deste trabalho, tiramos por conclusão que a vida do UNIX foi inicializada como um sistema de compartilhamento de tempo para microcomputadores, mas agora é usado em máquinas que vão do notebook até o supercomputador.
Na atualidade o UNIX é o sistema operacional que mais roda em arquiteturas de máquinas de qualquer outro sistema operacional conhecido.
O MS-DOS é um sistema de comandos enviados por teclado para o computador realizar as funções, sendo ele o sistema operacional da Microsoft que era utilizado antes da criação do Windows.
O MS DOS é utilizado por um motivo próprio, seja por comodidade com comandos digitados, para poder executar comandos mais avançados para o computador ou mesmo por saudosismo.
Esperamos que de forma fácil e compreensível tenhamos alcançado nosso objetivo, que era de passar para vocês o que o UNIX e o MS DOS representa para um Sistema Operacional.

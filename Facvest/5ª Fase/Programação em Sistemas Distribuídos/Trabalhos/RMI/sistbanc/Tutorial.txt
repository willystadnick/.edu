Tutorial para Execução Local:

 Abrir uma janela de comando e ir até o local onde foi salvo a aplicação
	cd \java\sistbanc
 Compilar as aplicações java 
	javac Banco.java
	javac BancoImpl.java
	javac Terminal.java
	javac Cliente.java
 Gerar os arquivos stub e skeleton para possibilitar o RMI
	rmic BancoImpl
 Iniciar o serviço RMIRegistry
	start rmiregistry
 Executar a classe servidor
	java BancoImpl
 Abrir outra janela de comando e ir até o local onde foi salvo a aplicação
	cd \java\sistbanc
 Executar a classe cliente
	java Terminal
 Navegar pelos menus do programa
 Finalizar a classe cliente (2ª janela de comando) atraves de comando interno
 Finalizar a classe servidor (1ª janela de comando)
	Ctrl + C
 Finalizar o serviço RMIRegistry
	Ctrl + C
 Fechar as janelas de comando abertas
	exit

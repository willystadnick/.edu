Quais os procedimentos usados para alterar a ordem de seqüência de boot dos sistemas operacionais para máquinas que possuem mais de um sistema operacional instalado.

R: no caso do gerenciador de boot utilizada no linux(grub), encontramos o arquivos menu.lst dentro do diretorio "/boot/grub/menu.lst" onde é feita a ordenação, e bem como a localização dos sistemas operacionais.




Para levantar um serviço de rede usa-se o comando "service nome_do_serviço start". Existe algum procedimento usado para carregar automaticamente todos os serviços de rede? Como este procedimento é realizado?

R: em distribuições do tipo Red Hat, como a Connectiva, é usado no shell do sistema o comando ntsysv. Nele podemos selecionar os serviços a serem inicializados automaticamente durante o boot do sistema.




Os serviços de rede podem ser instalados e atualizados através dos CD's de instalação do CL10 (3 CD's). No caso de uma rede, como por exemplo, o laboratório de redes com 40 computadores, o professor teria que disponiblizar 120 CD's, iou seja 3 CD's para cada aluno. Isto seria tedioso e praticamente inviável. Qual alternativa isentaria o professor de reprozir dúzias de CD's? Observe que esta alternativa deve ser efetiva.

R:no caso das distribuições connectiva, é usado pacotes rpm de instalação de programas e aplicativos, e a lista de localizacão desses pacotes se encontram em " /etc/apt/source.list". E os comandos para a atualização e instalação é apt-get install "Nome do Pacote".

Criar usuários e privilégios de leitura, escrita e execução. Modificar permissões (chmod) dos usuários de forma que os mesmos não possam alterar arquivos. Testar os procedimentos.

R: Usando o comando adduser -g root, criaremos usuários com provilégios superusuario(root).

Sobre o FTP:

a) Para que serve o FTP e em que situações um usuário o utilizaria? 

R:o FTP (File Transfer Protocol) é um protocolo de troca de arquivos pela rede. A transferência de arquivos localizados em computadores remotos é feito através de um software cliente, que se conecta no servidor FTP. Ele é utilizado para transferir documentos como softwares, textos, imagens e sons.




b) Como o FTP é usado no servidor e no cliente?

R: o FTP é utilizado no servidor 

c) Configurar o FTP para usuário aluno. Testar através de máquinas linux (browser e konqueror) e windows (browser). Observe que o FTP funciona através do konqueror com o seguinte comando ftp://usuario@IP. E também através do browser linux/windows com o seguinte comando ftp://usuario@IP.

d) Configure o servidor FTP para permitir 1 e 10 conexões. Testar as funcionalidades.

e) Configurar o FTP para ser acessado via usuário anonymous.
